# **App Name**: Cyclonapp

## Core Features:

- Auth and User Onboarding: Secure sign-up and sign-in functionality capturing user name, email, and preferred income frequency (biweekly or monthly).
- Intelligent Budget Allocation Tool: AI-powered planning tool that dynamically recalculates the 50/30/20 rule based on income and real-time debt burdens, prioritizing necessities and reducing savings allocations when debts exceed limits.
- Supabase Central Database: Integration with Supabase to manage user profiles, debt histories, fixed costs (like Netflix), and payment records via relational tables.
- Comprehensive Debt Management: Full CRUD functionality for debts featuring proximity alerts, amount tracking, and due dates with clear visual indicators.
- Smart Payment Interface: A streamlined modal for marking debts as 'Paid' with separate flows for full installment confirmation or custom partial payment values.
- Savings Growth Tracker: Visual dashboard summarizing total accumulated savings and a period-by-period history of the 'self-pay' distributions.
- Proactive Payment Reminders: Logical scheduling for payment alerts such as 'Upcoming' or 'Due Now' to ensure financial health.

## Style Guidelines:

- The primary color is a soft lavender (#8C7DE6) representing trust and organizational clarity.
- Background follows the lavender hue with a highly desaturated tint (#FBFBFF) for a clean, light financial dashboard environment.
- Accent color is a harmonious periwinkle (#8096E6) to differentiate secondary actions from the main flow while maintaining a professional look.
- Supplemental palette includes pastel mint green for growth, sky blue for peace, and soft pink for emphasis, ensuring no neon tones are used.
- Headline and body font: 'Poppins' for a contemporary, precise, and avant-garde aesthetic that feels approachable for personal finance.
- Use rounded, thin-stroke icons that emphasize the 'organized and friendly' style guide of the application.
- A minimal Bottom Tab Navigator ensures essential navigation between Gestión, Deudas, and Ahorro remains accessible at all times.
